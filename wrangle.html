About The Project

WRANGLE is an app for locating businesses in the Normal Heights neighborhood of San Diego. It starts with an optional tutorial on how the app works and follows up with a log in/sign up screen. From there, the user can see local businesses and check in to a location by scanning a QR code. The user is awarded points for supporting local businesses and these points can then be redeemed for rewards.


This app is based off of a book I created that documents various artworks in the Normal Heights neighborhood of San Diego.

The application is intuitve and follows the same color scheme as the book to create a cohesive product from print to web. After using the book as a reference point, the application translated to mobile in a way that allows the user to support local businesses in the Normal Heights neighborhood in an interactive way.

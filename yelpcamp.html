About The Project

This full-scale web application allows users to add new campgrounds to a database. Users can also comment on existing campgrounds and delete their previous posts. In order to post, users must sign up and log in. The user interface is intuitive and straight forward with limited frills for straightforward user experience.

The application is minimal and easy to navigate. The user personas made during initial testing lended themselves to clean design and the user testing performed concluded that simplistic sans-serif and blue buttons were perfered over red buttons and serif typeface. The database set up lends itself to repeat visitors and continuous interactions.
